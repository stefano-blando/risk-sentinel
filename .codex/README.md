# Codex CLI Prompt Templates

Prompt pronti per delegare task meccanici a Codex CLI.

## Usage
```bash
codex "$(cat .codex/prompt_name.md)"
```

## Available Prompts
- `data_wrangling.md` — Convert PhD correlation data to NetShock format
- `test_stubs.md` — Generate test stubs for core modules
- `plotting.md` — Standard network visualization templates
