# RiskSentinel — Teleprompter Script (2 Minutes)

Use this as a read-aloud script while recording.

## 0:00 - 0:10 | Website intro
Action:
- Open the project website.
- Scroll to the animated network section.

Read:
"RiskSentinel is an agentic systemic-risk simulator for real-time financial contagion analysis.  
It combines deterministic network simulation with AI-assisted synthesis to support fast, auditable decisions."

## 0:10 - 0:20 | Switch to app
Action:
- Switch to the Streamlit app tab.

Read:
"Now I’ll run the core application flow."

## 0:20 - 0:40 | One-click crisis replay
Action:
- Sidebar -> `Crisis Presets` -> click `COVID-19 Crash`.

Read:
"With one click, we replay a real stress window and load the corresponding market network.  
The platform computes contagion dynamics, affected institutions, and cascade depth."

## 0:40 - 0:55 | Model comparison
Action:
- Click `Compare All 3 Models`.

Read:
"We can compare propagation models side by side, so risk conclusions are not tied to a single modeling assumption."

## 0:55 - 1:30 | Agentic query
Action:
- In chat/demo input, run this prompt:

`Compare systemic risk between JPM and GS on 2025-12-01 using DebtRank, highlight cascade depth and sector concentration, then propose a hedging plan.`

Read:
"Here, deterministic simulation provides the quantitative backbone, and the agent layer adds structured synthesis and risk actions."

## 1:30 - 1:50 | Explainability and verification
Action:
- Open the explainability/verification area.

Read:
"Outputs are backed by explicit diagnostics and verification checks, so recommendations remain transparent and reproducible."

## 1:50 - 2:00 | Closing line
Action:
- Keep the key metrics/results visible.

Read:
"RiskSentinel turns shock scenarios into fast, explainable, and decision-ready systemic-risk intelligence."

## Backup line (if needed)
